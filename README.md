# Optimal Siting and Sizing of Virtual Power Lines
Online supplement accompanying the paper on Optimal Siting and Sizing of Virtual Power Lines.

We have the following files
- online_supplement.pdf: Additional results and validation
- milp_gurobipy.py: Gurobipy modl of the MILP described in the paper
- grids/ with all the used grids in a power-grid-model json format.
